# To-Do List Drag and Drop Application

This project is a simple To-Do List application that allows users to create, manage, and organize their tasks using drag-and-drop functionality. 

## Features

- Add and remove to-do items
- Drag and drop to reorder tasks
- Simple and intuitive user interface

## Project Structure

```
todo-drag-drop
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Styles for the application
│   ├── js
│   │   ├── app.js        # Main JavaScript file
│   │   └── dragdrop.js   # Drag-and-drop functionality
│   └── components
│       └── todoItem.js   # TodoItem class definition
├── .gitignore            # Files to ignore in Git
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd todo-drag-drop
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage

1. Open `src/index.html` in your web browser.
2. Add new tasks to the list.
3. Drag and drop tasks to reorder them as needed.
4. Remove tasks by clicking the delete button next to each item.
